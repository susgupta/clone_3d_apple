# Sushil G Apple Dynamic Three JS and GSAP Clone

This is a clone of Apple's iPhone 15 Pro website using React.js and TailwindCSS. It highlights the effective use of GSAP (Greensock Animations) and Three.js for displaying iPhone 15 Pro models in various colors and shapes.

This is design/code exercise to explore
- Animations using GSAP
- Scroll effects to showcase a product
- Adding interactive 3D models

First acknowledgements:

- iPhone based on "Apple iPhone 15 Pro Max Black" by **polyman** (https://sketchfab.com/3d-models/apple-iphone-15-pro-max-black-df17520841214c1792fb8a44c6783ee7) 

- Credit YouTube channel **JS Mastery** for inspiration and tutorial (https://www.youtube.com/@javascriptmastery)
